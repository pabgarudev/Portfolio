## Code

Inline code like `npm run dev` sits in a sentence. A fenced block gets syntax highlighting and its own frame:

```ts
interface Post {
  title: string;
  date: Date;
  tags: string[];
}

function isPublished(post: Post & { draft: boolean }) {
  return !post.draft;
}
```

## Image with caption

For a captioned image (the same treatment every project page uses), this post needs to be an `.mdx` file (not `.md`), then import the shared `Figure` component and an image from `src/assets`:

```mdx
import Figure from "../../components/Figure.astro";
import myImage from "../../assets/blog/my-post/photo.png";
```

```jsx
<Figure
  src={myImage}
  alt="Description for screen readers"
  caption="The text shown under the image."
/>
```
